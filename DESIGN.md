# VEZvision Design System

## 1. Brand feel

VEZvision should feel like a focused software house / digital studio: modern, precise, premium, calm, and credible. The site must be confident without pretending to be a large agency or showing fake scale.

## 2. Color tokens

- Page background: `#f5f5f5` / transparent grid surfaces.
- Primary text: `#101828`.
- Secondary text: `#475467` and `#667085`.
- Muted borders: `#eaecf0`, `#d0d5dd`.
- Primary action: `#101828`, hover `#1d2939`.
- Surface: `#ffffff`, soft surface `#f5f5f5`.

## 3. Typography

- Primary font: Satoshi-style sans-serif already used by the site.
- Headings: tight tracking, medium-to-semibold weight, clear hierarchy.
- Body copy: concise, practical, readable at 14–16px with relaxed line height.

## 4. Spacing and layout

- Base rhythm: 4px increments.
- Content containers: 1200px max width with 24px side padding.
- Sections should breathe, but avoid empty-looking gaps on mobile.
- Mobile layouts collapse to one column and must not create horizontal overflow.

## 5. Components

- Hero sections use video/visual atmosphere with one H1 and clear CTA.
- Cards use soft neutral surfaces, subtle borders/shadows, and restrained motion.
- Empty states are intentional product moments, not database errors.
- CTA buttons are dark, compact, rounded, and direct.

## 6. Motion

- Motion is supportive only: reveal, subtle hover lift, skeleton shimmer.
- Animate `transform` and `opacity`; avoid layout animation.
- Respect `prefers-reduced-motion`.

## 7. Content principles

- Write honestly for the current stage of the company.
- Do not invent portfolio items, client scale, or inflated claims.
- Prefer concrete language about problems, process, delivery, and outcomes.
- Avoid generic AI/SaaS clichés and overpromising.
